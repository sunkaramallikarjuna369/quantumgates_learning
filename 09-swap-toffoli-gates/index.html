Toffoli Gate (CCNOT)

The Toffoli gate (also called CCNOT or Controlled-Controlled-NOT) is a three-qubit gate. It flips the target qubit only when BOTH control qubits are |1⟩.

Toffoli as Classical AND Gate

The Toffoli gate can implement classical logic reversibly. With the target initialized to |0⟩, the output is the AND of the two controls.

Universal for Classical Computing

The Toffoli gate is universal for classical reversible computing. Any classical Boolean function can be implemented using only Toffoli gates!

Key Properties

SWAP Properties

SWAP² = I

Self-inverse: swapping twice returns to original.

SWAP = 3 CNOTs

CNOT·CNOT·CNOT

Decomposable into three CNOT gates.

Toffoli Properties

CCNOT² = I

Self-inverse: applying twice returns to original.

Classical Universal

AND, OR, NOT

Toffoli can implement any classical Boolean function.
