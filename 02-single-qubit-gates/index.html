Single-Qubit Gates

Rotations on the Bloch Sphere

Prerequisites

1. Qubit State Representation

A qubit state |ψ⟩ = α|0⟩ + β|1⟩ is a point on the Bloch sphere. The amplitudes α and β are complex numbers with |α|² + |β|² = 1.

2. Matrix Multiplication

Gates are 2×2 matrices that multiply state vectors. The result is a new state vector representing the transformed qubit.

3. Unitary Matrices

All quantum gates are unitary: U†U = I. This ensures probability is preserved and operations are reversible.

Which Gate Should I Use?

Flip |0⟩ to |1⟩: Use X gate - quantum NOT
Create superposition from |0⟩: Use H gate - creates |+⟩ = (|0⟩+|1⟩)/√2
Add 180° phase to |1⟩: Use Z gate - multiplies |1⟩ by -1
Add 90° phase to |1⟩: Use S gate - multiplies |1⟩ by i
Add 45° phase to |1⟩: Use T gate - multiplies |1⟩ by e^(iπ/4)
Flip AND add phase: Use Y gate - combines X and Z

General Single-Qubit Gate Parameterization

Any single-qubit gate can be written as U(θ, φ, λ) with three parameters:

θ (theta)

Rotation angle (0 to π). Controls how far the state moves from the poles.

φ (phi)

Azimuthal angle of rotation axis. Controls the direction of rotation.

λ (lambda)

Phase parameter. Adds a global phase to the transformation.

U(θ, φ, λ) = Rz(φ) · Ry(θ) · Rz(λ)

Any gate = sequence of Z and Y rotations!
