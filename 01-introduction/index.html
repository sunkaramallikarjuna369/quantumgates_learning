Before We Begin - Essential Prerequisites

1. Classical Bits - The Foundation

A classical bit is the simplest unit of information - it can only be 0 or 1. Think of it like a light switch: it is either OFF (0) or ON (1), never in between. All classical computers work by manipulating billions of these bits.

2. Logic Gates - How Bits Transform

Classical gates (AND, OR, NOT) transform bits. The NOT gate flips: 0 to 1 and 1 to 0. Quantum gates are similar but work on qubits and can do much more - they can create superpositions!

3. Vectors - Representing States

We represent states as column vectors. |0⟩ = [1, 0]ᵀ means "100% probability of measuring 0". |1⟩ = [0, 1]ᵀ means "100% probability of measuring 1". The numbers represent probability amplitudes.

4. Matrix Multiplication - How Gates Work

Gates are 2x2 matrices. When we multiply a gate matrix by a state vector, we get a new state. This is how quantum gates transform qubits!

Understanding Quantum Gates - The 4W+H Approach

WHAT is a Quantum Gate?

A quantum gate is a unitary operation that transforms a qubit's state. Mathematically, it's a 2x2 matrix (for single-qubit gates) that multiplies the state vector. Visually, it's a rotation on the Bloch sphere. Every quantum gate preserves the total probability (|α|² + |β|² = 1) and is reversible.

WHY do we need Quantum Gates?

Quantum gates are the building blocks of quantum algorithms. Just like classical computers use AND, OR, NOT gates to compute, quantum computers use quantum gates. The key difference: quantum gates can create superposition (being in multiple states at once) and entanglement (correlating qubits). This enables quantum speedup for certain problems like factoring large numbers (Shor's algorithm) and searching databases (Grover's algorithm).

WHEN should you use Quantum Gates?

X gate: When you need to flip |0⟩ to |1⟩ (quantum NOT). H gate: When you need to create superposition from a basis state. Z gate: When you need to add a phase to the |1⟩ component. CNOT: When you need to entangle two qubits or implement conditional logic.

WHERE do Quantum Gates fit in Quantum Computing?

Quantum gates form quantum circuits. A circuit reads left-to-right: qubits start in |0⟩, pass through gates, and are measured at the end. Gates are the "instructions" that manipulate quantum information. Any quantum algorithm can be built from a small set of universal gates (like H, T, and CNOT).

HOW do Quantum Gates work?

Gates work through matrix-vector multiplication. If your qubit is in state |ψ⟩ = [α, β]ᵀ and you apply gate U (a 2x2 matrix), the new state is U|ψ⟩. On the Bloch sphere, this corresponds to a rotation. For example, the X gate rotates 180° around the X-axis, swapping |0⟩ and |1⟩.

Key Insight: Superposition

A qubit in superposition |ψ⟩ = α|0⟩ + β|1⟩ is NOT "either 0 or 1 but we don't know which." It's genuinely in BOTH states simultaneously until measured. The coefficients α and β are complex numbers called probability amplitudes. When measured, the probability of getting 0 is |α|² and getting 1 is |β|². This is fundamentally different from classical probability!

Understanding θ and φ - The Complete Picture

θ (Theta) - The Polar Angle

θ measures the angle from the +Z axis (north pole). It ranges from 0° to 180°:

  • θ = 0°: State is at +Z (north pole) = |0⟩
  • θ = 90°: State is on the equator (superposition states like |+⟩, |-⟩, |i⟩, |-i⟩)
  • θ = 180°: State is at -Z (south pole) = |1⟩
|ψ⟩ = cos(θ/2)|0⟩ + e^(iφ)sin(θ/2)|1⟩

Notice: We use θ/2 in the formula! This is because the Bloch sphere represents states up to a global phase, and the actual state space is a 2D complex sphere.

φ (Phi) - The Azimuthal Angle

φ measures the angle around the Z-axis in the XY plane. It ranges from 0° to 360°:

  • φ = 0°: Points toward +X axis (|+⟩ state when θ=90°)
  • φ = 90°: Points toward +Y axis (|i⟩ state when θ=90°)
  • φ = 180°: Points toward -X axis (|-⟩ state when θ=90°)
  • φ = 270°: Points toward -Y axis (|-i⟩ state when θ=90°)

φ represents the relative phase between |0⟩ and |1⟩ components. This phase is invisible when measuring in the Z-basis but becomes important in quantum interference!

Cartesian Coordinates from Angles

The Bloch vector coordinates (x, y, z) relate to angles by:

x = sin(θ)cos(φ), y = sin(θ)sin(φ), z = cos(θ)

These are standard spherical coordinates! The state vector always has length 1 (it's on the unit sphere surface).

WHY Does the X Gate Matrix Look Like This?

Step 1: What should X do?

The X gate should swap |0⟩ and |1⟩. So we need: X|0⟩ = |1⟩ and X|1⟩ = |0⟩

Step 2: Translate to vectors

In vector form: X[1,0]ᵀ = [0,1]ᵀ and X[0,1]ᵀ = [1,0]ᵀ

Step 3: Build the matrix

For a 2x2 matrix [[a,b],[c,d]], we need:

  • First column = where |0⟩ goes = [0,1]ᵀ, so a=0, c=1
  • Second column = where |1⟩ goes = [1,0]ᵀ, so b=1, d=0
