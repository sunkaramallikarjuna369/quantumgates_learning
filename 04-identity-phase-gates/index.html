Identity & Phase Gates

I, S, and T Gates for Phase Manipulation

Prerequisites

1. What is Phase?

In quantum mechanics, phase is the angle of a complex number. For |ψ⟩ = α|0⟩ + β|1⟩, the relative phase between α and β affects interference but NOT measurement probabilities.

2. Complex Numbers

e^(iθ) = cos(θ) + i·sin(θ). When we multiply by e^(iπ/2) = i, we rotate by 90°. When we multiply by e^(iπ) = -1, we rotate by 180°.

3. Z-axis on Bloch Sphere

Phase gates rotate around the Z-axis. They don't change |0⟩ or |1⟩ probabilities, but they change the position on the equator (superposition states).

Which Phase Gate Should I Use?

Add 180° (π) phase to |1⟩: Use Z gate - flips sign of |1⟩
Add 90° (π/2) phase to |1⟩: Use S gate - multiplies |1⟩ by i
Add 45° (π/4) phase to |1⟩: Use T gate - smallest standard phase
Transform |+⟩ to |i⟩ (Y-axis): Use S gate - rotates from +X to +Y
Transform |+⟩ to |-⟩: Use Z gate - rotates from +X to -X
Universal quantum computation: Need T gate + Clifford gates (H, S, CNOT)

Why T Gate is Special

Universal Quantum Computation

The Clifford group {H, S, CNOT} can create many quantum states but cannot perform universal quantum computation alone. Adding the T gate makes the set universal - you can approximate any quantum operation!

T Gate in Algorithms

The T gate is used extensively in:

  • Quantum error correction (magic state distillation)
  • Shor's algorithm for factoring
  • Grover's search algorithm
  • Quantum simulation of molecules
