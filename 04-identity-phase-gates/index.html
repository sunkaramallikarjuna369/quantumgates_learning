Phase Gates: S and T

Fine-Grained Phase Control - The Key to Universal Quantum Computing

Prerequisites - Understanding Phase

1. What is Phase?

In quantum mechanics, phase is the "angle" of a complex number. For a state α|0⟩ + β|1⟩, the relative phase between α and β affects interference. Phase is invisible when measuring in the Z-basis but crucial for quantum algorithms!

2. Complex Exponentials

e^(iθ) = cos(θ) + i·sin(θ). Key values: e^(iπ) = -1, e^(iπ/2) = i, e^(iπ/4) = (1+i)/√2. These are the phases added by Z, S, and T gates respectively.

3. The Z Gate

Z adds a phase of e^(iπ) = -1 to |1⟩. This is a 180° rotation around the Z-axis. S and T are "fractional" versions of Z: S = √Z (90°), T = ⁴√Z (45°).

4. Why Phase Matters

Phase determines interference patterns. When paths recombine (like after H gates), phases add or cancel. This is how quantum algorithms achieve speedup - constructive interference on correct answers, destructive on wrong ones!

Phase Gates - The 4W+H Framework

WHAT are Phase Gates?

Phase gates add a phase factor to the |1⟩ component while leaving |0⟩ unchanged. They have the form [[1,0],[0,e^(iθ)]] where θ is the phase angle. Z (θ=π), S (θ=π/2), and T (θ=π/4) are the most common.

WHY do we need different phase gates?

Different algorithms need different phase precision. Z gives coarse 180° control. S gives 90° control (needed for many circuits). T gives 45° control and is essential for universal quantum computing - the set {H, T, CNOT} can approximate ANY quantum operation!

WHEN to use each phase gate?

Z: Phase kickback, Z-basis phase flip. S: Quarter-turn phases, preparing |i⟩ state. T: Fine phase control, magic state distillation, fault-tolerant computing. S†, T†: Reverse phases (rotate the other way).

WHERE in quantum circuits?

Phase gates appear after creating superposition (H gate). They modify the relative phase before interference. In fault-tolerant computing, T gates are the "expensive" gates - circuits are often optimized to minimize T-count.

HOW do they work on the Bloch sphere?

Phase gates rotate around the Z-axis. Z rotates 180°, S rotates 90°, T rotates 45°. States on the Z-axis (|0⟩, |1⟩) are unchanged (eigenstates). Equatorial states rotate around the equator.

S Gate (Phase Gate / √Z)

WHAT does S do?

S adds a 90° phase (factor of i = e^(iπ/2)) to |1⟩. S|0⟩ = |0⟩, S|1⟩ = i|1⟩. On the Bloch sphere, it's a 90° rotation around the Z-axis. S² = Z, so S is the "square root of Z".

WHY is the matrix [[1,0],[0,i]]?

Step-by-step derivation:

  1. S|0⟩ = |0⟩ → first column is [1,0]ᵀ
  2. S|1⟩ = i|1⟩ = i[0,1]ᵀ = [0,i]ᵀ → second column is [0,i]ᵀ
  3. Combining: S = [[1,0],[0,i]]

The factor i = e^(iπ/2) represents a 90° phase rotation.

HOW does S transform |+⟩?

|+⟩ = (|0⟩ + |1⟩)/√2. Applying S:

S|+⟩ = (S|0⟩ + S|1⟩)/√2 = (|0⟩ + i|1⟩)/√2 = |i⟩

The state moves from +X axis to +Y axis on the Bloch sphere!

Key Insight: S creates the |i⟩ state

Starting from |+⟩ = (|0⟩+|1⟩)/√2, applying S gives |i⟩ = (|0⟩+i|1⟩)/√2. This is the +Y eigenstate! Similarly, S|-⟩ = |-i⟩. The S gate rotates between X-basis and Y-basis states.

T Gate (π/8 Gate / ⁴√Z)

WHAT does T do?

T adds a 45° phase (factor of e^(iπ/4)) to |1⟩. T|0⟩ = |0⟩, T|1⟩ = e^(iπ/4)|1⟩. On the Bloch sphere, it's a 45° rotation around the Z-axis. T² = S, T⁴ = Z, T⁸ = I.

WHY is the matrix [[1,0],[0,e^(iπ/4)]]?

Step-by-step derivation:

  1. T|0⟩ = |0⟩ → first column is [1,0]ᵀ
  2. T|1⟩ = e^(iπ/4)|1⟩ → second column is [0,e^(iπ/4)]ᵀ
  3. e^(iπ/4) = cos(π/4) + i·sin(π/4) = (1+i)/√2

WHEN is T essential?

The T gate is crucial for universal quantum computing. The gate set {H, T, CNOT} can approximate any quantum operation to arbitrary precision. In fault-tolerant quantum computing, T gates are the most "expensive" gates to implement, so algorithms are often optimized to minimize "T-count".

Key Insight: T makes quantum computing universal!

Without T (or similar non-Clifford gates), quantum computers could be efficiently simulated classically (Gottesman-Knill theorem). The T gate provides the "magic" that makes quantum computers powerful. This is why T gates are so important in quantum error correction and fault-tolerant computing.

Mathematical Verification

T² = [[1,0],[0,e^(iπ/4)]]² = [[1,0],[0,e^(iπ/2)]] = [[1,0],[0,i]] = S
S² = [[1,0],[0,i]]² = [[1,0],[0,i²]] = [[1,0],[0,-1]] = Z
Z² = [[1,0],[0,-1]]² = [[1,0],[0,1]] = I

Practical Applications

Creating Y-basis states

To create |i⟩ = (|0⟩+i|1⟩)/√2 from |0⟩:

|0⟩ → H → |+⟩ → S → |i⟩

Phase Kickback

When a controlled-phase gate acts on a superposition control qubit, the phase "kicks back" to the control. This is the basis of quantum phase estimation!

Quantum Fourier Transform

QFT uses controlled rotations with angles π/2, π/4, π/8, etc. These are implemented using controlled-S, controlled-T, and finer rotations.
